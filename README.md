# YDA (Yo?DA Engine) - Yes!! Another SDL Game Framework :)

## Requirements
1. C++11 Compiler
2. CMake 3.1
3. Patience :o)

## Install
### Windows: 
#### You should set the enviroments variables
1. SDL2DIR
2. SDLIMAGEDIR

#### If doesn't work, you should configure manually the following variables on CMake
1. SDL2MAIN_LIBRARY
2. SDL2_INCLUDE_DIR
3. SDLIMAGE_INCLUDE_DIR
4. SDLIMAGE_LIBRARY
